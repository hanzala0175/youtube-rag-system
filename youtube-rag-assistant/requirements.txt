# NOTE on the LangChain ecosystem: as of mid-2026 LangChain went through a
# major restructuring (v1.0) that moved most legacy chains/retrievers into a
# new `langchain-classic` package and archived `langchain-community` (no new
# releases, but still functional). The FAISS integration still lives in
# `langchain-community` — `langchain-classic` only re-exports it via a
# deprecated shim — so this project depends on `langchain-community`
# directly for FAISS, while building the actual RAG chain with plain LCEL
# runnables from `langchain-core` (which has the strongest backwards-compat
# guarantees) instead of any legacy chain class. This combination avoids the
# "class moved to another module" errors that plague older LangChain
# tutorials.
